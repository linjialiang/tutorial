# 概述

Gitea 是一个轻量级的 DevOps 平台软件。这里只针对安装和使用，具体说明请查阅[[👉官方文档]](https://docs.gitea.com/zh-cn/)

## 环境准备

1. 操作系统：Debian 12
2. 数据库：PostgreSQL >= 12
3. Git
4. Go：>= 1.22
5. node >= 18
6. make
7. Gitea：源码包

::: tip 提示
首次测试，我使用了 lnmpp 一键包来测试，PostgreSQL/Git/node 这些已经安装成功，做个配置即可
:::
