# 概述

PSR 即 `PHP Standards Recommendations`，中文翻译为 `PHP标准建议`；

除了 PSR 外，这里还会介绍与其同宗的 PER，PER 即 `PHP Evolving Recommendations`，中文翻译为 `PHP发展建议`；

截止 `2025/9/24` ，已通过提案的 PSR 有 `14` 种，已发布的 PER 有 `1` 种。
