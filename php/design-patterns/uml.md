## UML 类图

描述类、接口、协作及他们之间的关系的图。显示系统中类的静态结构。

::: info 有什么作用？
描述软件系统的静态结构 - 对系统的词汇建模 - 对简单协作建模 - 对逻辑数据库模式建模
:::

## 类图说明

::: info 是否抽象：

1. 类名为斜体则为抽象类
2. 类方法为斜体则为抽象方法

:::

::: info 矩形 3 层：

1. 顶层：类名称
2. 中间层：类属性
3. 底层：类方法

:::

::: info 属性/方法访问控制：

| 符号 | 说明                |
| ---- | ------------------- |
| '-'  | 表示私有(private)   |
| '#'  | 表示保护(protected) |
| '+'  | 表示公开(public)    |

:::
